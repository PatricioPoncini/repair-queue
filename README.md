## `📜 Commands`

### `🚀 Run Server`

```bash
make run
```

### `🔬 Run Tests`

```bash
make test
```

### `✨ Run Lint`

```bash
make lint
# or
make lint-fix
```

### `📦 Run Migrations`

`Up Migrations ⬆️`

```bash
make migrate-up
```

`Down Migrations ⬇️`

```bash
make migrate-down
```
